# Top 10 Movies Collection (Flask + API)

## Overview

A Flask web application for managing a personal top 10 movies list.

## Description

This project allows users to search for movies using an external movie database API, add them to a local SQLite database, and rate and review each movie.  
It demonstrates API integration, CRUD operations with SQLAlchemy, and form handling in Flask.
