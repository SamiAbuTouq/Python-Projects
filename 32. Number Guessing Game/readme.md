# Number Guessing Game (Flask + Python)

## Overview

A simple web-based number guessing game built with Flask.

## Description

When you open the home page, the app randomly picks a number between 0 and 9.  
You guess by visiting `/<number>` in the URL, and the app tells you if your guess is too high, too low, or correct, with a GIF response.
